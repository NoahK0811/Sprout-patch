# Sprout Patch

A tiny browser gardening game where you buy seeds, water plots, grow flowers, harvest them, and sell them at the market.

## How to play

Open `index.html` or `garden-game.html` in a browser.

## GitHub Pages

This folder is ready for GitHub Pages. Upload these files to a GitHub repository:

- `index.html`
- `garden-game.html`

Then enable GitHub Pages from the repository settings.
